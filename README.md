# webpack + director + art-template 实现 前后端分离的小的项目结构
1. yarn install
2. npm run dev xxx/xxx 运行项目开发环境
3. npm run build xxx/xxx 打包项目

### 主要功能
1. webpack 实现html页面生成，静态资源地址配置，js拆分打包等。
2. director 实现简单的前端路由访问。
3. art-template 实现前端模板渲染。
